# Matplotlib Basics

## What is Matplotlib?

Matplotlib is a Python library used for creating charts, graphs, and visualizations.

It is one of the most popular data visualization libraries in Data Science and Machine Learning.

## Why Use Matplotlib?

- Easy to create charts
- Helps visualize data trends
- Useful for data analysis
- Widely used in Data Science projects

## Installation

```python
pip install matplotlib
```

## Importing Matplotlib

```python
import matplotlib.pyplot as plt
```

## Simple Line Plot

```python
import matplotlib.pyplot as plt

x = [1, 2, 3, 4]
y = [10, 20, 30, 40]

plt.plot(x, y)
plt.show()
```

## Common Functions

- plt.plot() → Line Chart
- plt.bar() → Bar Chart
- plt.scatter() → Scatter Plot
- plt.hist() → Histogram
- plt.pie() → Pie Chart
- plt.show() → Display Chart
