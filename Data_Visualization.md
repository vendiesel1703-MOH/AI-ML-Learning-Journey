# Data Visualization Basics

## What is Data Visualization?

Data Visualization is the graphical representation of data using charts, graphs, and plots to understand patterns, trends, and insights.

## Why Data Visualization?

- Makes data easier to understand
- Helps identify trends and patterns
- Supports better decision making
- Improves communication of insights

## Popular Python Libraries

- Matplotlib
- Seaborn
- Plotly
- Pandas Visualization

## Common Charts

### Line Chart
Used to show trends over time.

### Bar Chart
Used to compare categories.

### Histogram
Used to show data distribution.

### Pie Chart
Used to show proportions.

### Scatter Plot
Used to show relationships between variables.

## What I Learned Today

Data Visualization helps transform raw data into meaningful visual insights using charts and graphs.
